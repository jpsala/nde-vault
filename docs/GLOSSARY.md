# Glossary

| Término | Significado |
| --- | --- |
| AOS | Agentic OS local: capa liviana de reglas, memoria, topics, tracks, skills y scripts para agentes. |
| NDE | Near Death Experience; experiencia cercana a la muerte. |
| ECM | Experiencia cercana a la muerte. |
| Vault | Carpeta `vault/`, fuente canónica Markdown/Obsidian del corpus. |
| Caso | Nota que representa una experiencia individual trazable a una fuente. |
| Concepto | Nota que representa una pieza del paradigma, sostenida por evidencia del corpus. |
| Raw transcript | Transcripción original preservada sin edición. |
| Readable transcript | Transcripción visible/canónica legible, derivada y auditable. |
| AUTO block | Bloque delimitado para contenido generado o actualizado por agentes/scripts. |
