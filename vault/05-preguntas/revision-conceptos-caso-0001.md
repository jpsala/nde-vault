## Cómo usar este documento

Este documento es la bandeja de revisión humana antes de crear o modificar conceptos.

Para cada concepto:

- marcar checkboxes;
- escribir comentarios debajo;
- decidir si se crea como concepto nuevo, se fusiona con otro, se deja como subconcepto o se descarta.

Leyenda sugerida:

- [ ] **Referenciar**: este caso debería aparecer como evidencia de ese concepto.
- [ ] **Crear**: crear nota nueva en `vault/02-conceptos/`.
- [ ] **Fusionar**: no crear nota separada; integrarlo en otro concepto.
- [ ] **Observar**: dejarlo registrado, pero no convertirlo aún en concepto.

> Nota: los links “transcripción” saltan al bloque aproximado dentro de la transcripción preservada. Los links “YouTube” abren el video en el timestamp.
