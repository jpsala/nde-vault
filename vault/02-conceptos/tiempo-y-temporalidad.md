# Tiempo y temporalidad

## Definición provisional

Familia conceptual para agrupar cómo las NDEs/ECMs describen el tiempo, la simultaneidad, la secuencia de eventos, la memoria, la revisión de vida y la relación entre tiempo ordinario y estados de conciencia no ordinarios.

## Lectura desde el paradigma NDE/ECM

En el Caso 0001, el tiempo aparece como algo menos fundamental que en la percepción cotidiana: durante la revisión de vida los eventos no se ordenan linealmente, la experienciadora afirma ver “derretirse” el tiempo y luego formula que el tiempo ordinario funciona como un constructo o plantilla para operar en la vida física. En el Caso 0002, el experienciador no describe una revisión de vida no lineal, pero sí una ausencia de tiempo medible durante la fase de luz/resonancia.

Este concepto marco permite acumular variantes futuras sin crear una nota aislada para cada matiz desde el primer caso: tiempo no lineal, ausencia de tiempo, simultaneidad, expansión o ralentización temporal, acceso a pasado/presente/futuro, y múltiples líneas temporales.

## Contraste con el paradigma clásico

El paradigma ordinario suele tratar el tiempo lineal como marco básico de experiencia: pasado, presente y futuro se ordenan de manera secuencial. Este concepto agrupa evidencia donde esa estructura aparece como relativa, funcional o dependiente del estado de conciencia.

## Síntesis acumulada

<!-- AUTO:SYNTHESIS:START -->
El concepto queda sostenido por los casos 0001 y 0002. La evidencia apunta a una temporalidad no ordinaria con variantes distintas: en el Caso 0001, revisión de vida no secuencial, relativización del tiempo lineal y aparición de múltiples líneas temporales como observación emergente; en el Caso 0002, ausencia de tiempo medible durante la fase de luz/resonancia, sin evidencia suficiente para clasificarlo como tiempo no lineal complejo.

## Tensiones / matices

- Mantener `tiempo-y-temporalidad` como paraguas, no como afirmación cerrada sobre la naturaleza última del tiempo.
- Separar ausencia de tiempo medible, simultaneidad, no-linealidad y múltiples líneas temporales como matices distintos.
- Separar el fenómeno experiencial (“no ocurre en tiempo lineal” / “no hay tiempo”) de interpretaciones posteriores más amplias (“el tiempo es un constructo”).
- `multiples-lineas-temporales` queda como subconcepto emergente hasta reunir más casos.
