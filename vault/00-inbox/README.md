# Inbox

Carpeta para URLs, notas rápidas, ideas sueltas y fuentes todavía no procesadas.

## Cómo pasar una URL al agente

En el chat, el usuario puede escribir algo como:

```txt
Procesá esta NDE:
https://www.youtube.com/watch?v=...
```

O:

```txt
Agregá esta fuente al corpus y detectá conceptos:
https://www.youtube.com/watch?v=...
```

El agente debe seguir `docs/workflow-url-a-caso.md`.

## Registro manual opcional

También se pueden dejar URLs pendientes en esta carpeta como archivos `.md`.
